# SHA1-CRACKER
SHA 1 Cracker Tools

# Installation
 ```sh
 $ git clone https://github.com/mnovel/SHA1-CRACKER
 ```

# USE
```sh
$ python run.py sha1 start end str
```

$ python2 run.py 595aa739fc58403c7c62cc2840d0b7fb 1 6 "0123456789 abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ"
